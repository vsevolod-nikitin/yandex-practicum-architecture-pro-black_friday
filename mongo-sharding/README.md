# pymongo-api

## Как запустить

1. Запускаем mongodb и приложение

```shell
docker compose up -d
```

2. Ожидаем запуск и инициализацию контейнеров. Проверяем, что контейнер configSrv имеет статус Healthy.
```shell
docker ps
```

3. Настраиваем шардирование. При ошибках выполнения повторяем вызов скрипта (через небольшое время).
```shell
./scripts/mongo-setup.sh
```

4. Заполняем mongodb данными

```shell
./scripts/mongo-init.sh
```

5. Для проверки заполнения шардов используйте команду

```shell
./scripts/mongo-verify.sh
```

## Как проверить

### Если вы запускаете проект на локальной машине

Откройте в браузере http://localhost:8080

### Если вы запускаете проект на предоставленной виртуальной машине

Узнать белый ip виртуальной машины

```shell
curl --silent http://ifconfig.me
```

Откройте в браузере http://<ip виртуальной машины>:8080
